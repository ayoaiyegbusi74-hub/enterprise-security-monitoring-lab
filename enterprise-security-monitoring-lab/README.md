
# Enterprise Security Monitoring Lab

> A complete enterprise-style Security Operations Center (SOC) lab built using Splunk Enterprise, Red Hat Enterprise Linux (RHEL 9), Kali Linux, Apache, and DVWA to simulate, detect, investigate, and document multi-stage web application attacks.

---

## Project Overview

The Enterprise Security Monitoring Lab was designed to simulate a real-world Security Operations Center (SOC) environment for monitoring, detecting, investigating, and documenting web application attacks.

The lab uses **Splunk Enterprise** as the Security Information and Event Management (SIEM) platform to ingest Apache web server logs generated by a vulnerable web application (DVWA). Simulated attacks from a Kali Linux attacker machine were detected through custom Splunk searches, alerts, threat hunts, correlation searches, and executive dashboards.

The project demonstrates the complete incident response lifecycle, from attack simulation to detection engineering, threat hunting, correlation, MITRE ATT&CK mapping, and incident investigation.

---

# Objectives

- Build an enterprise-style SOC monitoring environment.
- Simulate realistic web application attacks.
- Centralize Apache web logs into Splunk Enterprise.
- Develop custom detection rules and alerts.
- Perform proactive threat hunting.
- Correlate multiple attack stages into a single investigation.
- Map attacker behavior to the MITRE ATT&CK framework.
- Produce executive dashboards and incident documentation.
