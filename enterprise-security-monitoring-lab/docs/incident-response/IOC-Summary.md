
# Indicators of Compromise (IOC) Summary

## Overview

This document summarizes the Indicators of Compromise (IOCs) identified during the simulated multi-stage web application attack conducted against the DVWA environment.

The identified IOCs were extracted from Apache access logs and analyzed using Splunk Enterprise during detection engineering, threat hunting, and incident response activities.

# Analyst Assessment

The identified IOCs indicate a coordinated multi-stage attack originating from a single attacker IP.

The attack followed a logical progression from reconnaissance through exploitation and persistence, matching common attacker behavior observed during web application compromises.

The documented IOCs can be incorporated into future monitoring, detection rules, and incident response playbooks.
